# A股个股深度分析

## 触发条件

- 用户要求"深度分析XX公司"、"分析一下XX的基本面"、"XX的护城河是什么"
- 用户要求了解某只A股的经营范围+财务+近期涨跌原因
- 用户给出股票代码/名称，要求全面研究

## 数据获取策略

### 优先级1：问财 API（iwencai skills）

并行调用4个问财 skill 获取数据：
1. `iwencai-base` → 主营业务、经营范围、行业分类
2. `iwencai-company-data` → 营收、净利润、毛利率、ROE、负债率、现金流
3. `iwencai-market-data` → 股价、涨跌幅、成交量、换手率、估值
4. `iwencai-event-data` → 公告、新闻、研报、股东变动

### 优先级2：东方财富网页抓取（API 不可用时降级）

当问财 API 返回 401/无权限时，立即降级到网页数据源，不要反复重试。

关键页面：
```

https://quote.eastmoney.com/sh{code}.html  # 沪市
https://quote.eastmoney.com/sz{code}.html  # 深市

# 用 JS 提取行情表格数据

# 财务摘要（在行情页内嵌）
```

### 优先级3：delegate_task 并行搜索

当需要补充定性信息（护城河分析、行业地位、近期涨跌催化剂）时，用 delegate_task 并行搜索：
- Task 1: 公司概况 + 主营业务 + 竞争优势 + 行业地位 + 财务数据
- Task 2: 近期股价异动原因 + 利好利空 + 分析师观点
- Task 3: 最新行情 + 估值对比 + 技术面

注意 delegate_task 最多3个并发，合理合并任务。

## 分析报告结构

### 一、公司概况与经营范围
- 公司简介（一句话定位）
- 核心业务构成（表格：业务板块/收入/占比）
- 所属概念板块

### 二、护城河分析
- 逐条列出竞争优势（特许经营权/规模效应/品牌/技术/网络效应/转换成本）
- 护城河的局限性（同样重要，避免只说好话）

### 三、最新财务数据
- 核心指标表格：营收/净利润/毛利率/净利率/ROE/负债率/现金流
- 估值指标：PE/PB/股息率/市值
- 财务特征总结（2-3句话点评趋势）

### 四、近期涨跌原因
- 下跌/低迷原因（分条列出）
- 潜在利好/上涨催化（分条列出）
- 股吧情绪摘要（作为散户情绪温度计）

### 五、收入构成深度拆解（第二轮）
- 按产品分类表格：产品/营收/占比/毛利率/毛利贡献占比
- 按地区分类表格：地区/营收/占比/毛利率
- 关键发现（哪个业务真正赚钱，哪个是"伪高端"）

### 六、估值分析（第二轮）
- 当前估值指标：PE/PB/PS/市值
- 同行对比表格：3-5家可比公司的PE/PB/市值/净利率
- 三档情景测算：保守/中性/乐观的合理股价区间
- 反推隐含预期：当前股价需要多少净利润才能支撑

### 七、综合评价
- 2-3句话总结投资逻辑
- 短期关注点
- 长期风险/机会

## 两轮分析流程

用户偏好将分析分为两轮：

### 第一轮：纯数据收集（不夹杂记忆/历史上下文）
- 并行启动：东方财富行情页抓取 + delegate_task 搜索（3路并发）
- 输出第一轮报告：公司概况、护城河、财务数据、涨跌原因

### 第二轮：深度分析（结合记忆和交叉验证）
- 抓取 F10 经营分析页获取收入分拆（按产品/地区的营收+毛利率）
- delegate_task 搜索：同行估值对比、券商盈利预测、可转债/重组方案细节
- 估值分析：反推隐含预期、同行PE/PB对比、合理估值区间测算
- 更新报告文件并 git push

## F10 经营分析页（收入分拆金矿）

```
# 经营分析页 URL（含按产品/地区分类的收入+毛利率明细）
https://emweb.securities.eastmoney.com/pc_hsf10/pages/index.html?type=web&code={SZ或SH}{code}&color=w#/jyfx

# JS 提取全部文本（不在 iframe 中，可直接获取）
(() => {
  const txt = document.body.innerText;
  const ls = txt.split('\n').filter(l => l.trim().length > 0);
  const kw = ['按产品分类','按行业分类','按地区分类','特种','电力','毛利率','主营构成','经营评述'];
  return JSON.stringify(ls.filter(l => kw.some(k => l.includes(k))).slice(0, 50), null, 2);
})()
```

关键数据点：按产品分类的营收/占比/毛利率、按地区分类的营收/占比/毛利率、经营评述全文（含客户名单、新签订单、业务进展等一手信息）。

## 估值分析方法论

1. 反推隐含预期：当前市值 ÷ 合理PE = 市场隐含的净利润预期，对比实际净利润看透支程度
2. 同行对比：找3-5家同行业可比公司的PE/PB/市值/净利率，定位估值分位
3. 情景测算：保守（当前盈利）/ 中性（增长兑现）/ 乐观（新业务爆发）三档，各给合理PE，算出股价区间
4. 关键验证：特种/高端业务的实际毛利率是否匹配市场给的估值溢价（经常存在"伪高端"——名义上特种但毛利率与普通产品差距不大）

## 注意事项

- 东方财富行情页标题格式为 `{股票名} {价格} {涨跌额}({涨跌幅}%)最新价格_行情_走势图—东方财富网`，可直接从标题提取实时价格
- 东方财富 F10 经营分析页（#/jyfx）内容不在 iframe 中，browser_console JS 可直接提取，是获取收入分拆的最佳数据源
- 东方财富 F10 财务分析页（#/cwfx）可获取多季度财务趋势数据（毛利率/净利率/ROE/负债率等时间序列）
- 股吧帖子用 browser_console + JS 提取效率最高
- 分析时要区分"事实数据"和"推断观点"，推断部分需标注
- 用户偏好中文输出，估值分析要结合A股市场特征（不要套用美股标准）
- 警惕"伪特种"陷阱：某些公司标榜特种/高端，但实际毛利率与普通产品差距不大，不应给予高估值溢价
